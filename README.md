# 📊 Superstore Sales Analysis Dashboard | Power BI

## 📌 Project Overview

The **Superstore Sales Analysis Dashboard** is an interactive Business Intelligence solution built using **Microsoft Power BI**. The dashboard analyzes sales, profit, customers, products, and regional performance to help businesses make data-driven decisions.

The project transforms raw Superstore transactional data into meaningful visual insights through multiple interactive dashboards.

This dashboard enables users to:

- Monitor business performance
- Track sales and profit trends
- Analyze customer behavior
- Identify best-selling products
- Compare regional performance
- Measure growth over time
- Evaluate discount impact on profitability

# 📈 Business Insights

### Sales

- Total sales exceeded **$2.3 Million**
- West region generated the highest sales.
- Technology category contributed the highest revenue.

### Profit

- Overall profit reached **$286K**
- Profit margin remained around **12.47%**
- West region generated the highest profit.

### Customers

- Consumer segment contributed the largest share.
- Top customers generated significant revenue.
- Customer count peaked during the final quarter.

### Products

- Canon imageCLASS 2200 Advanced Copier generated the highest sales.
- Technology products dominated both revenue and profit.

### Regional Performance

- West region performed best overall.
- South region requires performance improvement.
